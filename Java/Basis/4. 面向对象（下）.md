### 4. 面向对象（下）



#### ==与equals

##### ==

**基本类型** 比较其数值是否相同

**对象** 比较其引用的地址是否相同

##### equals

所有类的equals方法均继承Object类的equals方法.其源代码:

```java
public boolean equals(Object obj) {
	return (this == obj);
}
```

效果与 == 相同
**因此,使用equals方法时,注意有没有被重写**
